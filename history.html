History

The 1930's

The idea of machine learning and neural networks first coined in the 1930's-1940's. This started with the first concepts outlined in the 1930's This was because of the recent advances in neuroscience.

The 1960's & 1970's

In the early 1960's the first "k-means" was first used for pulse code modulation by James MacQueen, J. B. (1967). Later in the 1970's the Multilayers NN & BP (Backpropagation) algorithms where devloped.

The 1980's

First time a BP (Backpropagation) method was applied to a neural network. This had many predecessors in the 1960's. In 1985 the first Bayesian Network was made. This operates on the Bayes Rule which was invented around 1700. In the late 1980's the Descision Tree method was created for Neural Networks.

The 1990's

Non-linear SVM & Clustering devloped. This includse supervised learning for classification and regression analysis.

2006

In 2006 the DBN (Deep Boltzmann machine) method was implemented which. This method is a type of binary pairwise used for hidden layers.

2009

In 2009 Neural networks start to get more widly used. This is known as the "Big bang" of deep learning.

2012

In 2012 AlexNet was the firs Neural Network to use GPU acceleration on a CNN (convolutional neural network ). This NetWork was used for image recognition and used techniques such as flipping images to train.

2014

The first implementation of VGGNet (Image recognition), Googlenet and GAN'S (Generative adversarial network).

2015

In 2015 a network with 4 layers was created and is known as "ResNet". This is a jump from AlexNets 8 layers to 152. This is also the year Tensoflow was created as a open source machine learning product.

2016-2017-2018

DenseNet was created in 2016 and was more efficant to Resnet. In 2017 WGAN was created as a more efficant extension version of a GAN. In 2018 DeepMind was used to diagnosing 50 differnet phthalmic conditions from 3D CT scans, this was know as a key barrier in the Machine leanring Field.
