Training Neural Networks

For a Neural Network to function correctly it needs to be trained. This process is similar if not exactly how a human learns something new. Training a Neural Network is based on two principals, supervised learning and unsupervised learning. Unsupervised learning allows the computer to use unlabelled data to find its own way to complete the final result. The alternative is supervised learning where the computer uses data that has been labelled data by a human and follows by example. There are many different ways to train a network following the Loss function, training function, Backpropagation each with their own mathematical formula.
